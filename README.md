# <img src="assets/sa-logo.svg" alt="Sreehari Arun Logo" width="100" /> Sreehari Arun - Architectural Portfolio

A minimalist, high-performance portfolio website for Sreehari Arun, an architect specializing in brutalist, minimal, and technical design. The site serves as a digital exhibition space, blending rigorous technical documentation with emotive spatial rendering.

![Project Preview](assets\projects\1\render_3.png)

## ğŸ›ï¸ Design Philosophy

The website embodies the architect's design ethos: **"Void & Form"**.
It utilizes a strict architectural grid system, varying between "Technical" (drawing/blueprint) and "Pictorial" (rendering/photo) modes, allowing users to explore projects through different lenses. But importantly, it maintains a clean, Swiss-style typographic hierarchy using `Space Grotesk` and `Inter`.

## âœ¨ Key Features

- **Dynamic View Toggle:** Switch instantly between 'Technical' (Line Drawings) and 'Pictorial' (Renderings) modes globally across the site.
- **Mosaic Grid Layout:** Asymmetric, responsive grid system that reflects architectural massing.
- **Responsive Design:** Fully optimized experience across Desktop, Tablet, and Mobile devices.
- **Performance Focused:** prioritized vanilla HTML/CSS/JS for maximum speed and minimal bloat.
- **Custom Branding:** Minimalist SVG logo and iconography designed specifically for this portfolio.
